== Description ==

**Tracking Revisions** is a very light plugin that lets you easily and safely tracking anything you want. All data will be pushed to a third party, Slack or Telegram. Deactivating this plugin brings your site back exactly to the state it was before.

This plugin is kindly brought to you by [Hien Nguyen Duy](https://www.linkedin.com/in/hiennguyenduy/) (please donate me ([CLICK HERE](https://www.paypal.com/paypalme/hiennguyenduy)))

This plugin is only maintained, which means we do not guarantee free support. Consider reporting a problem and be patient.

= Features =
* Tracking (Post / Page / Custom Post Type)
* Tracking Linkout

= Upcoming Features =
* Tracking (Taxonomy / Custom Taxonomy)
* Tracking Comments
* Tracking Settings
* Tracking Media
* Telegram Integration

= Support Languages =

== Installation ==

1. Go to Plugins › Add New.

2. Search for *Tracking Revisions*.

3. Look for this plugin, download and activate it.

4. You can change this option any time you want, just go back to Settings › Tracking Revisions.
